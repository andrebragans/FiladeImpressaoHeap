# Fila de Impressão com Prioridades

Para rodar:
1. Certifique-se de ter o Java instalado (`java -version`).
2. Execute o arquivo JAR:
   - Via terminal: `java -jar FilaImpressaoComPrioridade.jar`
  
Estrutura:
- Documento.java: representa um documento com prioridade.
- FilaImpressao.java: gerencia a fila usando PriorityQueue (heap).
- MainGUI.java: interface gráfica com Swing para interação com a fila.

*Urgentes são destacados com ⚠️ e cor vermelha.
